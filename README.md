# Big Mart Sales Prediction using XGBoost  
Regression Problem | Gradient Boosting | Machine Learning Project

This project focuses on predicting **product sales in Big Mart outlets** using the  
**XGBoost Regressor**, a powerful gradient boosting algorithm known for high performance  
on structured tabular data.

---

## Project Overview
- Load and explore the Big Mart sales dataset  
- Handle missing values and inconsistent categories  
- Perform feature engineering and encoding  
- Apply feature scaling (where required)  
- Split data into training and testing sets  
- Train an **XGBoost Regressor**  
- Evaluate model performance using regression metrics  

---

## Algorithm Used
### XGBoost Regressor
- Ensemble gradient boosting technique  
- Handles non-linear relationships effectively  
- Robust to multicollinearity  
- High performance on tabular datasets  

---

## Features Used
- Item Weight  
- Item Fat Content  
- Item Visibility  
- Item Type  
- Item MRP  
- Outlet Size  
- Outlet Location Type  
- Outlet Type  

Target Variable:
- **Item_Outlet_Sales**
